# Hand Gesture Recognition using MediaPipe

This project demonstrates real-time hand gesture and hand sign recognition using **MediaPipe**, **OpenCV**, and simple machine learning models.  
Hand pose is estimated using MediaPipe Hands, and gestures are classified using custom classifiers.
